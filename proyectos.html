PROYECTOS

Si te preguntas qué pasa cuando la creatividad se encuentra con el código, esta sección es la respuesta. Aquí plasmo mis ideas en proyectos tangibles, utilizando un abanico de herramientas que domino y gestionando cada paso con Git para un control de versiones impecable.

He utilizado HTML y CSS para dibujar interfaces y darles estilo, Java para construir lógicas y sistemas robustos, JavaScript para crear experiencias interactivas y dinámicas, y Python para inyectar inteligencia, automatización y manejo de datos. Cada proyecto es una pequeña obra, creada con la tecnología más adecuada para el desafío, y lista para ser explorada. ¡Adelante!

Proyecto 1: Pull & Bear

Aquí te presento mi trabajo en el desarrollo front-end de la web de Pull&Bear, donde me enfoqué en crear una experiencia de usuario fluida y visualmente atractiva para el dinámico mundo de la moda.

Proyecto 2: Jinx, la bala perdida

Aquí les presento mi fan page dedicada a Jinx de Arcane, donde me centré en replicar su energía caótica y su estética única a través de un diseño web dinámico y una experiencia de usuario envolvente. Fue un reto divertido traducir su personalidad explosiva en líneas de código.
