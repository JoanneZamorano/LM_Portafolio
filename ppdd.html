Política de Privacidad

Fecha de última actualización: 15 de julio de 2025

Bienvenido/a a mi portafolio web personal. Como desarrolladora web, valoro y respeto la privacidad de mis visitantes. Esta política de privacidad describe cómo se maneja la información en este sitio web.

1. Información que Recopilo

Dado que este es un **portafolio personal** y no una plataforma de comercio electrónico o un servicio que requiera interacción con el usuario (como formularios de registro, comentarios, etc.), generalmente **no recopilo información personal identificable** de los visitantes de forma directa. No utilizo cookies de seguimiento invasivas ni solicito datos como nombres, correos electrónicos o información de contacto.

Sin embargo, es posible que se recopile **información no personal** de forma automática a través de:

  • Registros del Servidor (Server Logs): Como la mayoría de los sitios web, mi servidor puede registrar automáticamente información básica sobre las visitas, como la dirección IP (anonimizada, si es posible), el tipo de navegador, la página de referencia, las páginas visitadas y la hora de la visita. Esta información se utiliza únicamente para fines de **mantenimiento, seguridad y análisis estadístico** general del sitio, ayudándome a entender cómo se utiliza el portafolio y a mejorar su rendimiento.
  • Herramientas de Análisis Web (Opcional): Si utilizo alguna herramienta de análisis web (por ejemplo, Google Analytics en su modo de privacidad mejorada o soluciones autoalojadas con anonimización de IP), lo haré con el objetivo de obtener estadísticas agregadas sobre el tráfico del sitio, nunca para identificar a usuarios individuales. En caso de usar Google Analytics, siempre activaré las opciones de anonimización de IP y no compartiré datos con otros servicios de Google.

2. Uso de la Información

La información no personal recopilada se utiliza exclusivamente para los siguientes propósitos:

  • Para **mantener y mejorar** el funcionamiento de este portafolio web.
  • Para **analizar tendencias** y estadísticas de tráfico (de forma agregada y anónima).
  • Para **garantizar la seguridad** del sitio web y protegerlo contra ataques.

Nunca venderé, alquilaré ni distribuiré tu información a terceros.

3. Enlaces a Sitios Web de Terceros

Mi portafolio puede contener enlaces a otros sitios web (por ejemplo, repositorios de GitHub, plataformas de desarrollo, redes sociales, etc.). Una vez que abandones mi sitio web a través de estos enlaces, **no tengo control** sobre esos otros sitios y no soy responsable de sus prácticas de privacidad. Te recomiendo leer las políticas de privacidad de cualquier sitio web que visites.

4. Seguridad de los Datos

Tomo medidas razonables para proteger la información recopilada contra el acceso no autorizado, la divulgación, la alteración o la destrucción. Sin embargo, ten en cuenta que ninguna transmisión de datos por Internet es 100% segura y no puedo garantizar la seguridad absoluta de la información.

5. Cambios en esta Política de Privacidad

Puedo actualizar esta política de privacidad ocasionalmente. Cualquier cambio se publicará en esta página con una nueva fecha de "última actualización". Te animo a revisar esta política periódicamente para mantenerte informado sobre cómo se protege tu información.

6. Contacto

Si tienes alguna pregunta sobre esta política de privacidad, no dudes en contactarme a través de la sección de contacto de mi portafolio o directamente a info.joazamorano@gmail.com

Gracias por visitar mi portafolio.
