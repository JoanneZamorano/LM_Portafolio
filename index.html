PROYECTOS

PasteleriaAPP

Este repositorio es un proyecto desarrollado como parte de la asignatura de Programación y Entornos de Desarrollo de DAM. Se trata de una aplicación de gestión de negocios para una pastelería, que permite gestionar clientes, productos, ventas e imprimir tickets. También visualiza datos de ventas y controla el stock de la pasteleria.

QuizMasterPy

Este repositorio es un proyecto de la asignatura MPO (Python) de DAM. Permite a los usuarios realizar cuestionarios tipo test. El programa muestra preguntas con múltiples opciones, corrige automáticamente las respuestas y proporciona un resumen detallado al finalizar.
